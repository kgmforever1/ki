# Personal Portfolio

This is a simple personal portfolio website created using HTML, CSS, and JavaScript. It includes sections for Home, Projects, and Contact, along with a navigation menu that allows users to jump to each section. The website is designed to showcase projects with dummy data and provides a clean and visually appealing layout.

## Project Structure

```
personal-portfolio
├── index.html      # The main HTML file containing the structure of the portfolio
├── style.css       # The CSS file for styling the portfolio
├── script.js       # The JavaScript file for interactive elements
└── README.md       # This README file
```

## Features

- **Home Section**: A brief introduction to the portfolio.
- **Projects Section**: Displays multiple project cards, each containing:
  - Title
  - Short description
  - Technology tags
  - Links to the projects
- **Contact Section**: A simple contact form for inquiries.

## Getting Started

To run the portfolio website locally, follow these steps:

1. Clone the repository or download the files.
2. Open the `index.html` file in your web browser.
3. Navigate through the sections using the menu at the top.
